# 背包九讲复习笔记

## 第1讲：01背包问题

- **场景**：每种物品仅有一件，可以选择放或不放。
- **状态定义**：`dp[j]` 表示容量为 `j` 时能获得的最大价值。
- **转移方程**：`dp[j] = max(dp[j], dp[j - w[i]] + v[i])`
- **核心点**：为了保证每件物品只选一次，**容量循环必须逆序**。

C++

```
for (int i = 1; i <= n; i++)
    for (int j = m; j >= w[i]; j--)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

## 第2讲：完全背包问题

- **场景**：每种物品有无限件。
- **状态定义**：同01背包。
- **转移方程**：`dp[j] = max(dp[j], dp[j - w[i]] + v[i])`
- **核心点**：为了允许物品重复放入，**容量循环必须正序**。

C++

```
for (int i = 1; i <= n; i++)
    for (int j = w[i]; j <= V; j++)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

## 第3讲：多重背包问题

- **场景**：每种物品有 $C_i$ 件。
- **复习策略**：掌握**二进制拆分**。将 $C_i$ 拆分为 $1, 2, 4, \dots, 2^k, C_i - (2^{k+1}-1)$，转化成 01 背包处理。
- **复杂度**：$O(V \cdot \sum \log C_i)$。

## 第4讲：混合背包问题

- **场景**：有的物品只有一件，有的无限，有的有限。
- **做法**：在物品循环内进行判断：
  - 如果是 01，用逆序循环。
  - 如果是完全，用正序循环。
  - 如果是多重，用二进制拆分或单调队列。

## 第5讲：二维费用的背包问题

- **场景**：每件物品受到两种限制（如重量和体积）。
- **状态定义**：`dp[j][k]` 表示容量为 `j`、体积为 `k` 时的最大价值。
- **做法**：多嵌套一层循环，两个限制维度的循环方向均参考 01 背包（逆序）。

## 第6讲：分组背包问题

- **场景**：物品被分为若干组，每组中最多只能选一件。
- **核心逻辑**：先选组，再选容量，最后遍历组内物品。
- **注意**：**容量循环必须在组内物品循环之外且逆序**，否则会变成每组选多件。

C++

```
for (int i = 1; i <= group_count; i++)
    for (int j = V; j >= 0; j--)
        for (auto& item : group[i])
            if (j >= item.w) dp[j] = max(dp[j], dp[j - item.w] + item.v);
```

## 第7讲：有依赖的背包问题

- **场景**：选某个物品必须先选它的父节点（树形结构）。
- **做法**：**树形DP**。对每个节点，先递归处理子节点，然后利用分组背包的逻辑将子树的信息合并到父节点。

## 第8讲：泛化物品

- **定义**：物品的价值不是常数，而是关于容量的函数 $f(v)$。
- **合并逻辑**：两个泛化物品合并相当于求卷积：$f(v) = \max(f_1(v-h) + f_2(h))$。

## 第9讲：背包问题问法的变化

- **输出方案**：记录 `g[i][j]` 表示状态是从哪个前驱转移过来的。
- **方案总数**：将 `max` 改为 `sum`，`dp[0] = 1`。
- **最优方案总数**：额外开一个数组 `cnt[j]` 记录达到 `dp[j]` 的方案数。
- **第 K 优解**：将 `dp[j]` 扩展为 `dp[j][k]`，转移时合并两个有序序列。
