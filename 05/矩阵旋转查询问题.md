# 矩阵旋转查询问题

### 1. 题目背景

小灰灰拿到了一个 $n \times n$ 的方阵 $a$，其初始值由行号 $i$ 和列号 $j$ 决定（坐标从 1 开始）：

$$a_{i,j} = (i - 1) \times n + j$$

### 2. 操作描述

对该方阵进行 $q$ 次操作，每次操作包含四个参数：`op x y len`。

- **(x, y)**：子矩阵左上角的坐标。
- **len**：子矩阵的边长。
- **op = 1**：将该子矩阵进行一次 **顺时针旋转 90°**。
- **op = 2**：将该子矩阵进行一次 **逆时针旋转 90°**。

### 3. 查询任务

在所有 $q$ 次操作完成后，给定 $d$ 个查询。每个查询给出一个最终状态下的坐标 $(dx_i, dy_i)$，输出该位置上最终的数值。

------

### 4. 核心解题思路：逆向溯源法

由于 $n$ 和 $q$ 的范围很大，直接模拟旋转会导致时间复杂度爆炸（TLE）。

**正确逻辑：** 1. 记录所有的操作指令。

2. 对于每一个查询点 $(dx, dy)$，从**最后一次操作**开始往前倒推。
3. 如果当前点在某个旋转操作的范围内，就计算它在旋转**前**所在的坐标。
4. 所有的操作回溯完毕后，得到的坐标即为初始坐标，代入初始公式计算即可。

### 5. 坐标变换公式（基于相对坐标）

设当前点在大矩阵中的坐标为 $(dx, dy)$，子矩阵左上角为 $(x_0, y_0)$，边长为 $len$。

先求出相对坐标：$u = dx - x_0, \ v = dy - y_0$。

- **情形 A：当前操作是顺时针（op=1）**

  逆向推回（即逆时针转回）的公式：

  - $dx_{new} = x_0 + (len - 1 - v)$
  - $dy_{new} = y_0 + u$

- **情形 B：当前操作是逆时针（op=2）**

  逆向推回（即顺时针转回）的公式：

  - $dx_{new} = x_0 + v$
  - $dy_{new} = y_0 + (len - 1 - u)$



```c++
#include <iostream>
#include <vector>

using namespace std;

// 使用 long long 防止 (i-1)*n + j 计算溢出
typedef long long ll;

struct Operation {
    int op, x, y, len;
};

int main() {
    // 优化输入输出效率
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, q;
    if (!(cin >> n >> q)) return 0;

    // 存储所有旋转操作
    vector<Operation> ops(q);
    for (int i = 0; i < q; ++i) {
        cin >> ops[i].op >> ops[i].x >> ops[i].y >> ops[i].len;
    }

    int d;
    cin >> d;
    while (d--) {
        int qx, qy;
        cin >> qx >> qy;

        // 从最后一个操作开始倒着推
        for (int i = q - 1; i >= 0; --i) {
            int x0 = ops[i].x;
            int y0 = ops[i].y;
            int L = ops[i].len;

            // 判断查询点是否在当前旋转的子矩阵范围内
            if (qx >= x0 && qx < x0 + L && qy >= y0 && qy < y0 + L) {
                int u = qx - x0; // 相对行偏移
                int v = qy - y0; // 相对列偏移

                if (ops[i].op == 1) { 
                    // 顺时针逆推（逆时针旋转 90 度）
                    qx = x0 + (L - 1 - v);
                    qy = y0 + u;
                } else { 
                    // 逆时针逆推（顺时针旋转 90 度）
                    qx = x0 + v;
                    qy = y0 + (L - 1 - u);
                }
            }
        }

        // 计算初始位置对应的数值
        // 注意：使用 1LL 强制转换为 long long 防止溢出
        cout << (ll)(qx - 1) * n + qy << "\n";
    }

    return 0;
}
```
