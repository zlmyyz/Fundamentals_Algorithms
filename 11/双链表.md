你说的对，但是"UNO！"是一款……后面忘了。

不过菜菜子为了让这个常玩的纸牌游戏更加有趣，自定义了一系列的规则：

1. 每次游戏从 1 号玩家开始，共 $n$ 名玩家，他们按顺序顺时针坐在一张圆桌上进行游戏。最开始按**顺时针**依次出牌。第 $i$ 号玩家手里初始有 $a_i$ 张牌。
2. 纸牌分为 4 种，分别是普通牌（`C`）、禁止牌（`S`）、反转牌（`R`）和罚抽两张牌（`D`）。普通牌并没有效果；禁止牌可以让你的一名玩家（简称下家）暂停出牌一次；反转牌可以使当前出牌的顺序翻转，即接下来轮到你的上家出牌；罚抽两张牌可以使你的下家的牌数 $+2$，并且暂停出牌一次。所有牌都没有任何出牌的限制，玩家可以随意出任意一种牌。
3. 如果一名玩家手里的牌打光，该玩家会立即退出对局。

菜菜子知道了这把游戏中的出牌结果，请你告诉他每位玩家最后剩下几张牌。

### Input

第一行包含两个整数 $n, m$ ($2 \le n, m \le 2 \times 10^5$)，如题意所描述。

第二行包含 $n$ 个正整数 $a_i$ ($1 \le a_i \le 2 \times 10^5$)。

第三行包含一个长度为 $m$ 的字符串，表示这把游戏所有玩家的出牌结果。其中 `C` 为普通牌，`S` 为禁止牌，`R` 为反转牌，`D` 为罚抽两张牌。保证出牌顺序合法，即直到最后也有至少两位玩家有牌。

### Output

输出包含 $n$ 行，第 $i$ 行为第 $i$ 名玩家最终的手牌数量。

### Example

| **standard input** | **standard output** |
| ------------------ | ------------------- |
| 3 6 3 2 3 SRDCCD   | 3 0 3               |

### Note

首先一号玩家出禁止牌，二号玩家被禁止一回合，三号玩家出反转牌，使得出牌顺序反转，接下来轮到二号玩家出牌，打出 +2 牌使得一号玩家的手牌数量 +2，并使其禁止一回合。接下来三号、二号玩家各出一张普通牌，此时二号玩家手牌打完，一号玩家打出 +2 牌使三号玩家手牌数量 +2。最后三人的手牌数量分别为 3 0 3。

```
#include<bits/stdc++.h>
using namespace std;
const int N = 2e5 + 10;
// L[i]记录i左边的玩家，R[i]记录i右边的玩家
int a[N], L[N], R[N];

// 当玩家出局时，将其从双向链表中摘除
void remove_player(int u) {
    L[R[u]] = L[u]; // 右边玩家的左手，牵住当前玩家的左手
    R[L[u]] = R[u]; // 左边玩家的右手，牵住当前玩家的右手
}

int main() {
    // 优化输入输出速度
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    if (!(cin >> n >> m)) return 0;
    
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        // 初始化链表，0的左边是n-1，n-1的右边是0
        L[i] = (i - 1 + n) % n;
        R[i] = (i + 1) % n;
    }
    
    string s;
    cin >> s;
    
    int curr = 0; // 当前出牌人下标
    int dir = 1;  // 出牌方向：1 表示顺时针(看R数组)，-1 表示逆时针(看L数组)
    
    for (int i = 0; i < m; i++) {
        char card = s[i];
        a[curr]--; // 打出一张牌
        
        int u = curr; // 记录打出这张牌的玩家
        
        // 如果手牌打光，立马出局 (从链表中移出)
        if (a[u] == 0) {
            remove_player(u);
        }
        
        // Lambda函数：根据当前方向，获取玩家 p 挨着的下一个存活玩家
        auto get_next = [&](int p) {
            return dir == 1 ? R[p] : L[p];
        };
        
        // 结算各种牌的效果：
        if (card == 'C') {
            curr = get_next(u);
        } else if (card == 'S') {
            // 跳过下一个玩家，直接到下下个
            curr = get_next(get_next(u));
        } else if (card == 'R') {
            dir = -dir; // 永久反转方向
            curr = get_next(u);
        } else if (card == 'D') {
            int target = get_next(u); // 找到下家
            a[target] += 2;           // 下家罚抽两张
            curr = get_next(target);  // 下家被跳过，下下家出牌
        }
    }
    
    for (int i = 0; i < n; i++) {
        cout << a[i] << "\n";
    }
    
    return 0;
}
```
