# P16236 [蓝桥杯 2026 省 B] LQ 聚合

## 题目描述

2056 年，探险队在月球背面的环形山深处发现了一座信号发射塔，其核心控制台正在不断闪烁着一串长度为 $N$ 的粒子序列。

序列中的每个位置被严格定义为 $L$ 型粒子、$Q$ 型粒子，或者因岁月侵蚀而模糊不清的未知状态 $?$。这些粒子将被依次射入反应场，而反应场的稳定性取决于序列的“$LQ$ 聚合”数量，该数量被定义为所有满足 $1 \le i < j \le N$ 且第 $i$ 个节点为 $L$、第 $j$ 个节点为 $Q$ 的二元组数量。

为了重启这座沉睡的巨塔，探险队需要将序列中所有的 $?$ 修复为确定的 $L$ 或 $Q$。

现在，请你计算出在所有可能的修复方案中，所能得到的“$LQ$ 聚合”数量的最大值是多少。

## 输入格式

第一行输入一个整数 $N$，表示粒子序列的长度。

第二行输入一个长度为 $N$ 的字符串，仅包含字符 $L$、$Q$ 和 $? $，表示当前探测到的粒子序列状态。

## 输出格式

输出一个整数，表示在将所有 $?$ 替换为 $L$ 或 $Q$ 后，能获得的最大“$LQ$ 聚合”数量。

## 输入输出样例 #1

### 输入 #1

```
5
??L??
```

### 输出 #1

```
6
```

## 说明/提示

### 【样例说明】

一种最优的策略是将序列修复为 LLLQQ。此时位于前面的 $3$ 个 $L$ 与位于后面的 $2$ 个 $Q$ 共可产生 $3 \times 2 = 6$ 个聚合。

### 【评测用例规模与约定】

对于 $30\%$ 的评测用例，字符串 $?$ 的个数不超过 $10$。

对于所有评测用例，$2 \le N \le 10^5$。



```
#include<bits/stdc++.h>
using namespace std;
typedef long long LL;
const int N=1e5+10;

int main(){
    int n;
    cin>>n;
    string s;
    cin>>s;
    
    // 个人习惯：给字符串开头塞个空格，让真正的字符从下标 1 开始，方便和日常思维对齐
    s=" "+s;
    
    // res: 历史最高总分（也就是我们要的最终答案）
    // cur: 当前状态下的总分
    // q: 当前位置【右边】还剩下多少个 Q (或者被当成 Q 的 ?)
    // l: 当前位置【左边】已经攒了多少个 L (或者已经变成 L 的 ?)
    LL res=0, cur=0, q=0, l=0;
    
    // =========================================================
    // 第一步：【初始状态构建】
    // 我们先做个极端假设：把所有的 '?' 全都当成 'Q'。
    // 从右往左扫一遍，算出这个初始状态下，到底能得多少分。
    // =========================================================
    for(int i=n; i>=1; i--){
        if(s[i]=='Q' || s[i]=='?'){
            q++; // 只要是 Q 或者 ?，统统当成 Q，右边的 Q 数量加 1
        } 
        else if(s[i]=='L'
            // 如果遇到了真正的 L，它就能和它右边所有的 Q 组合
            // 所以当前总直接加上右 Q 的总数
            cur+=q; 
        }
    }
    
    // 把初始得分作为我们保底的最高分
    res=cur;
    
    // =========================================================
    // 第二步：【动态扫线，步步算账】
    // 我们拿着一个扫描仪，从左往右扫。
    // 遇到 '?' 就尝试把它从 'Q' 翻转成 'L'，看看是赚了还是亏了。
    // =========================================================
    for(int i=1; i<=n; i++){
        if(s[i]=='L'){
            // 遇到了固定的 L。它已经被扫描仪扫过了，正式加入左侧的 L 阵营
            l++;
        }
        else if(s[i]=='Q'){
            // 遇到了固定的 Q。它被扫描仪跨过去了，说明它不再属于“右边的 Q”了
            q--;
        }
        else if(s[i]=='?'){
            // 核心高光时刻：遇到了 '?'！我们现在要把它从 Q 变成 L。
            
            // 1. 它变身了，所以“右边的 Q”阵营里少了一员大将
            q--;
            
            // 2. 算经济账（得失差值）：
            // 赚到的：它变成了 L，就能和剩下所有的 q 组合，所以赚了 q 分
            // 亏掉的：它不再是 Q 了，原本能和它组合的所有左边的 l 都失效了，所以亏了 l 分
            // 净收益 = 赚的 - 亏的 = q - l
            cur += q - l;
            
            // 3. 看看这次变身后的总分，有没有打破历史最高记录
            res = max(res, cur);
            
            // 4. 变身彻底完成！它现在是一个堂堂正正的 L 了，加入左侧的 L 阵营
            l++;
        }
    }
    
    // 循环结束，res 里存的就是我们试探过的所有情况里的绝对最高分！
    cout<<res<<'\n';
    return 0;
}
```
